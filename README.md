
## Spring MVC 使用示例

> 通过简单的示例演示Spring MVC的强大功能。

### 项目内容

* @Controller注解使用
* Mapping请求
* 获取请求数据
* 生成响应
* 消息转换器
* 渲染视图
* 数据类型转换
* 校验机制
* 表单标签
* 文件上传
* 异常处理

### 运行项目

```
    $ cd spring-mvc-example
    $ mvn tomcat7:run [-Dmaven.tomcat.port=<port no.>] （默认是8080）
```

访问下面地址：

http://localhost:port/spring-mvc-example/
